# Portfolio Design Decisions

## Theme

- **Mode**: Dark mode default
- **Background**: Dark navy `#0a192f`
- **Accent**: Cyan/green accent colors

## Layout

- **Style**: Bento grid layout (2024-25 trend)
- Modular, card-based sections with varying sizes

## Animations

- **Approach**: Minimal animations
- Fade-ins on scroll/load
- Hover effects only
- No heavy transitions or complex motion

## Typography

- **Primary font**: Inter / Geist (sans-serif)
- **Code font**: JetBrains Mono (monospace)

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
